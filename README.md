# Zuber
Ride-Sharing Analysis

## Project Description
This project analyzes ride-sharing data for Zuber to identify patterns in trip demand 
and evaluate how external factors such as weather conditions impact ride frequency.

## Tools Used
- Python
- SQL
- Pandas
- Matplotlib / Seaborn

## Key Analysis
- Analysis of trip volume by company
- Impact of weather conditions on ride frequency
- Hypothesis testing related to ride demand
- Data-driven business insights

## Conclusion
The analysis demonstrates how weather and operational factors influence ride demand, 
supporting strategic decisions based on statistical evidence.
